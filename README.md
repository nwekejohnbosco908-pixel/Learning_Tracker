# lerning tracker
A simple and responsive learning tracker, web application that helps you track your daily learning progress.
Built with HTML, CSS, and Javascript, it allows users to log tasks, mark them as completed, and visually see their progress

## Live Demo

## Tech Stack
- HTML - structure of the website 
- CSS - styling and responsive design
- JAVASCRIPT - Functionality and dynamic updates

 ## Features
 - Add new learninf tasks
 - Mark tasks as completed
 - Delete tasks when finished
 - Track your learning progress visually
 - Responsive design for destop and mobile
